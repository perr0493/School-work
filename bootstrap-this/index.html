What Is Bootstrap?

Bootstrap is used to create responsive websites without any use of CSS, it's like a giant libary of css that uses javascript to do the work without you even knowning it. But just remember just like css you should work on mobile first and with bootstrap its even easier with its awesome powerful mobile-first flex grid to build layouts of all shaps and sizes. with all this power how could anything go wrong ?.

Where do I start with Bootstrap ?

First things first you need to open your coding software I use visual studio code. Then make a new page called it index.html from there you will need to link the bootstrap so go to https://getbootstrap.com/docs/5.0/getting-started/introduction/ and there under CSS you will find a 'link' which you will fit nicely into your head as a link. After this your ready to get Bootstraping !

Once you have done the linking your ready to get going to the races. the best way to start off is cover your mobile-first. Because bootstrap covers girnds and flex very well which makes it easier to scale up elements and optimize them for in all screen sizes. once you have made sure everything is up scaling the way you want you can add all the flare you want.
